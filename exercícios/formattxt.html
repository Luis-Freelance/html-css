Principais formatações

negrito / destaque

Nesta frase temos um negrito usando a tag b(não semantica)

Nesta frase, tenho um destaque usando a tag strong(semantica)

itálico/ ênfase

Nesta frase, temos um termo em itálico usando a tag i(não semantica)

Nesta frase,temos um termo em ênfase usando a tag em(semantica)

Texto marcado

Podemos criar também um texto marcado usando a tag mark.

Texto grande e texto pequeno

Estamos criando um texto grande com a tag big(não semantica) e um texto pequeno com a tag small(semantica), oque não tem sentido

Texto deletado

Podemos marcar um texto como excluído com a tag del para indicar que ele deveser lido mas não considerado

Texto inserido

Podemos marcar um texto como inserido com a tag ins para dar uma enfase e indicar que ele foi adicionado depois

Existe também o sublinhado com a tag u que é não semantica

Texto sobrescrito

Escrevemos coisas como x2+3 usando a tag sup

Texto subscrito

Escrevemos coisas como H20 usando a tag sub
